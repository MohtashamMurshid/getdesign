Four phases. About ninety seconds.

A coordinator agent orchestrates four sub-agents inside a Daytona sandbox. Crawl is deterministic. Extraction is deterministic. Only synthesis touches the LLM, and only against grounded tokens.

PHASE 01

Crawl

Fetch HTML, all linked CSS, @import chains, font sources.

PHASE 02

Screenshot

Real Chromium in a Daytona sandbox. Hero + full-page stitch.

PHASE 03

Extract

Deterministic CSS parse into a DesignTokens Zod object.

PHASE 04

Synthesize

GPT-5.5 class via AI Gateway. Structured output, validated.
