The web surface at [getdesign.app](https://www.getdesign.app) is a streaming chat UI with a live `design.md` artifact panel. Paste a URL and the extractor streams tokens into Markdown you can copy.

## When to use it

- You want a one-off design.md for a quick look.
- You want to compare two sites interactively.
- You want the polished chat experience without installing anything.

## What you get

- A streaming Markdown artifact with the full nine-section design.md.
- A live palette preview as colors are extracted.
- One-click copy and download.
