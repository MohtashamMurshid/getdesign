## Install

```bash
skills add MohtashamMurshid/getdesign
```

## Use it

> Use the getdesign skill on https://linear.app and write the result to `linear.design.md`.

Codex will render the page with its own browser tool and produce the `design.md` locally. No credentials leave your machine.
