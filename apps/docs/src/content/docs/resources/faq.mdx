## Does getdesign scrape HTML?

No. It renders the target URL in a real browser and reads computed CSS. Results reflect what users actually see.

## Can I use the output commercially?

The output describes publicly visible design choices of the target site. Respect the target's trademarks and terms. getdesign's own code is open source; see the [repository](https://github.com/MohtashamMurshid/getdesign).

## Which surface should I use?

- [Web](/surfaces/web/) to explore.
- [API](/surfaces/api/) to integrate server-side.
- [CLI](/surfaces/cli/) for scripts.
- [SDK](/surfaces/sdk/) inside a TypeScript app.
- [Skill](/surfaces/skill/) to run inside your own coding agent.

## Is there authentication?

Not in v1. Rate limits apply.

## Is there a free tier?

Yes. The beta is free. Private beta early access is collected on the [home page](https://www.getdesign.app).
