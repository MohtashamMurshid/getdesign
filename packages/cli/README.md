# @getdesign/cli

> The design system for any URL.

**Coming soon.** Private beta — join the waitlist at [getdesign.app](https://getdesign.app).

## Preview

```bash
npx @getdesign/cli cursor.com
npx @getdesign/cli linear.app --out design.md
```

getdesign opens a site in a real browser, extracts palette, typography, and
components, and returns a production-grade `design.md` — grounded in the site's
actual CSS.
